# Mangni – Community Matrimonial App

Mangni is a community-based real-time matrimonial platform that helps users to explore profiles find suitable matches within thier community.

## Features
- User profile creation
- View/ Edit own profile
- Admin approval system
- Gender-based browsing
- Search and filter
- Secure Supabase backend
- Easy User Interface
- Deployed to playstore

## Tech Stack
- React + TypeScript
- Supabase
- Capacitor (Android)
- Tailwind CSS

## How to Run
npm install
npm run dev
